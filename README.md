# Silverstripe SEO Module

Simple Module to adjust SEO properties on a page by page basis

## Installation

Installation via composer

```bash
$ composer require toastnz/toast-seo
```

Then add the following to the HEAD of your document 

```
    <% include ToastSEO %>
```
